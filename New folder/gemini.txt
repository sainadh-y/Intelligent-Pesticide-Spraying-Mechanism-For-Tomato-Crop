# Proposed Spray Calculation Formula

This document outlines a revised formula for the spray calculation in Phase 5. The goal is to replace the abstract, ratio-based formula with one that is grounded in more specific, standard agricultural parameters, making it clearer and more adaptable.

---

## 1. The Proposed Formula

The amount of spray in milliliters (`spray_ml`) for a single plant is calculated as follows:

```
spray_ml = (application_rate_l_ha / 10) * (1 / plant_density_sqm) * infection_severity_factor
```

---

## 2. Parameters and Factors

### `application_rate_l_ha`
- **Description**: The recommended application rate of the final, diluted spray product, measured in **Liters per Hectare**. This value is commonly found on the labels of commercial agricultural spray products.
- **Impacting Factors**:
    - **Product Choice**: Different fungicides or pesticides have different recommended concentrations and application rates.
    - **Target Disease/Pest**: The rate can change depending on the disease being treated.
    - **Weather Conditions**: Wind and rain can affect how much spray is needed.
    - **Equipment**: The type of nozzle and sprayer pressure influences the volume applied.
- **Reference Value (Default)**: A general-purpose starting value of **400.0 L/ha** is proposed. This represents a typical high-volume spray rate for vegetables like tomatoes to ensure thorough foliage coverage. This value should be configurable.
- **Supporting Documentation**: General guidelines for high-volume spraying on specialty crops often fall in the 300-600 L/ha range. The specific product label is always the primary authority.
    - *(See Reference 1 for various product-specific examples)*

### `plant_density_sqm`
- **Description**: The number of tomato plants cultivated per **square meter**.
- **Impacting Factors**:
    - **Tomato Variety**: Bushy (determinate) vs. vining (indeterminate) varieties have different space requirements.
    - **Cultivation Method**: Greenhouse, open field, staked, or caged planting all affect density.
- **Reference Value (Default)**: A default density of **2.5 plants/m²** is proposed. This is a common density for indeterminate tomato varieties grown with support in greenhouses or commercial fields.
- **Supporting Documentation**:
    - *(See Reference 2, which suggests 3-4 plants/m² for medium-height varieties)*.
    - *(See Reference 3, which indicates an average of 3.6 plants/m² for glasshouse conditions)*.

### `infection_severity_factor`
- **Description**: A multiplier that adjusts the base spray volume based on the `infection_average` (calculated in Phase 4). It allows the system to apply more or less spray depending on how sick the plant is.
- **Impacting Factors**: The sole factor is the `infection_average` score from the previous pipeline phase.
- **Reference Value (Default Tiers)**:
    - `infection_average <= 0.10`: **0.8** (Low infection, reduced spray)
    - `0.10 < infection_average <= 0.30`: **1.0** (Medium infection, standard spray)
    - `infection_average > 0.30`: **1.2** (High infection, increased spray)
- **Supporting Documentation**: This tiered approach is a common strategy for variable rate application in precision agriculture. The specific thresholds and factors are adjustable based on field observations and should be considered a starting point for tuning. This principle is discussed in concepts like "volume-based spray schedules" where application changes with plant condition.
    - *(See Reference 4 for an example of adjusting spray based on plant size/condition)*.

---

## 3. Formula Derivation

The formula converts a per-area rate into a per-plant amount.

1.  **Convert Application Rate to ml/m²**:
    - `rate_ml_sqm = application_rate_l_ha / 10`
    - (Since 1 L = 1000 ml and 1 ha = 10,000 m², the conversion is 1000/10000 = 1/10)

2.  **Calculate Area Per Plant**:
    - `area_per_plant_sqm = 1 / plant_density_sqm`

3.  **Calculate Base Spray Per Plant**:
    - `base_spray_ml = rate_ml_sqm * area_per_plant_sqm`

4.  **Apply Severity Adjustment**:
    - `final_spray_ml = base_spray_ml * infection_severity_factor`
